## Development
This app using prisma for query builder. and sqlite for database.
- [Prisma Docs](https://www.prisma.io/docs/) 

From your terminal:

```sh
# install all dependencies package
npm install
# create database
npx prisma push
# seeding data to database 
# /prisma/seed.ts
npm run seed
# run app in development mode
npm run dev
```
